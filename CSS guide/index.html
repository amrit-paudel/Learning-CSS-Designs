The important css properties and tricks

CSS background properties

                 

                    <body>
                      <div class="box" >

                      </div>
                    <body/>

                    .box {
                       background-image: url();
                       background-repeat: no-repeat;
                       background-attachment: fixed;
                       background-position: center;
                       background-size: contain;
                    }

The first image is when background-size: contain; and the second image is when background-size: cover; . In contain the image tries to contain within box. It means if the box is way too big than image than image won't stretch to fill whole box. But if the box size is smaller than image size than image will try to contain within box no matter what Whereas in cover the image tries to fill the whole box, here image may stretch or expand.

CSS position properties

                 

                    .any class name {
                      position: static;
                       position: relative;
                       top: 100px;
                    }

The first one position: static; is the normal default property of css boxes.
Whereas in the second one the css box moves 100px below from top relative to it's normal position 

                 

                    .any class name {
                       position: fixed;
                       top: 100px;
                    }

The first one position: fixed; here the css box moves 100 px below from the top with relative to viewport. 

                 

                    .any class name {
                       position: absolute;
                       top: 100px;
                    }

The first one position: absolute; here the css box moves 100 px below from the top with relative to it's nearest ancestors.
But that ancestors must be positioned non statically.
Meaning that ancestors cannot have position: static;
It must have other position properties like position: relative;

Otherwise on doing position: absolute; box will be positioned with respect to document body position: fixed; vs position: absolute; In position: fixed; box remains fixed and page scrolls
Whereas in position: absolute; box attached to page and scrolls with page

See an example

                 

                    <body>
                      <div class="box1" >
                      < p>
                      This is a box 1 
                      </p>

                      <div class="box2" >

                     }
